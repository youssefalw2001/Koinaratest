# Workspace

## Overview

pnpm workspace monorepo using TypeScript. This is **Koinara** — a Telegram Mini App with a dual-currency economy.

## Product: Koinara

Dark cyber-fintech Telegram Mini App targeting MENA market.

### Dual-Currency Economy
- **🔵 Trade Credits (TC)** — engagement-only, non-withdrawable. Used to place predictions. Earned via quests, daily rewards.
- **🪙 Gold Coins (GC)** — real value, withdrawable via USDT TRC-20. Earned by winning trades at 0.85x payout ratio.

### Core Features
- 60-second BTC/USDT predictions (Long/Short), 0.85x GC payout
- VIP subscription ($4.99/wk or $14.99/mo via TON, or 500 TC/week)
- VIP perks: higher daily GC cap (6,000 vs 800), max bet 5,000 TC (vs 1,000), better withdrawal rate, VIP-only quests, +50% streak TC bonus
- Free daily GC cap: 800 GC
- Min bet: 50 TC | Max bet: 1,000 TC free / 5,000 TC VIP
- Daily login reward: 100-150+ TC (streak-based)
- Quest system: TC rewards for social/exchange tasks
- Referral program
- Withdrawal queue: GC → USDT TRC-20 (free and VIP tiers)
- Leaderboard ranked by lifetime GC earned

### Withdrawal Queue
- Free tier: 4,000 GC = $1 | $2.50 min | 48–72hr | $25/week cap | $1.99 one-time verification fee (0.02 TON)
- VIP tier: 2,500 GC = $1 | $1 min | ~4hr | $100/week cap | No verification fee
- Both tiers: 2.5% platform fee on all withdrawals
- 50%-of-previous-day revenue daily operator payout cap
- Weekly cap resets every Monday

### Design System

- Gold (#f5c518) — Gold Coins, VIP
- Magenta (#ff2d78) — loss states, danger

## Stack

- **Monorepo tool**: pnpm workspaces
- **Node.js version**: 24
- **Package manager**: pnpm
- **TypeScript version**: 5.9
- **API framework**: Express 5
- **Database**: PostgreSQL + Drizzle ORM
- **Validation**: Zod (`zod/v4` in lib packages, `zod` in api-server), `drizzle-zod`
- **API codegen**: Orval — run `pnpm --filter @workspace/api-spec run codegen` to regenerate from `lib/api-spec/openapi.yaml`
- **Build**: esbuild (CJS bundle)

## Key Commands

- `pnpm run typecheck` — full typecheck across all packages
- `pnpm run build` — typecheck + build all packages
- `pnpm --filter @workspace/api-server run dev` — run API server locally
- `pnpm --filter @workspace/api-spec run codegen` — regenerate API client from openapi.yaml
- `pnpm --filter @workspace/db run push` — sync DB schema to database

## DB Tables

- `users` — dual-currency user fields (tradeCredits, goldCoins, totalGcEarned, vipExpiresAt, dailyGcEarned, hasVerified, weeklyWithdrawnGc, etc.)
- `predictions` — trade history (payout field = GC earned)
- `quests` + `quest_claims` — TC-reward quest system
- `withdrawal_queue` — GC withdrawal requests with full tier/fee/status tracking
- `platform_daily_stats` — daily revenue/payout tracking for 50%-of-prev-day daily payout cap
- `ad_watches` — ad watch tracking
- `gem_inventory` — Gem Shop powerups
- `vip_tx_hashes` — TON VIP payment deduplication
